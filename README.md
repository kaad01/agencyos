# AgencyOS

An open-source project management starter for consulting agencies: simple, aesthetic, and focused on the essentials — customers, colleagues, projects, budgets, capacity, and account health.

The first version is a polished React/Vite dashboard inspired by the clarity of agency ERP tools like MOCO: fewer screens, strong hierarchy, fast overview, and calm daily operations.

## Features

- **Project management**: active projects, status, budget, progress, deadlines, project lead.
- **Colleague management**: team roles, current focus, capacity/load overview.
- **Customer management**: accounts, revenue, open projects, relationship health.
- **Agency dashboard**: utilization, pipeline value, customer count, today’s focus.
- **Open-source friendly**: MIT license and clean Vite/React foundation.

## Getting started

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
